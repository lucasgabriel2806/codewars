# Converter string para camel case

## Descrição:

Complete o método/função para que ele converta palavras separadas por hífen (-) ou underline (_) em camel case.
A primeira palavra do resultado deve ser capitalizada somente se a palavra original também for capitalizada (conhecido como Upper Camel Case, também chamado de Pascal Case).
As próximas palavras devem ser sempre capitalizadas.

## Exemplos:

<code>"the-stealth-warrior"</code> é convertido para <code>"theStealthWarrior"</code>

<code>"The_Stealth_Warrior"</code> é convertido para <code>"TheStealthWarrior"</code>

<code>"The_Stealth-Warrior"</code> é convertido para <code>"TheStealthWarrior"</code>
