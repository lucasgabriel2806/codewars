# String ends with ?

<p>Complete a solução de forma que ela retorne verdadeiro se o primeiro argumento (uma string) passado terminar com o segundo argumento (também uma string).</p>

## Exemplos:

```
Entrada: "abc", "bc"
Saída: true

Entrada: "abc", "d"
Saída: false
```
