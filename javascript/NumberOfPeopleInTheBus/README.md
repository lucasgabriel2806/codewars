# Number of People in the Bus

Há um ônibus circulando pela cidade, parando em cada ponto de ônibus para pegar e deixar passageiros.

Você recebeu uma lista (ou array) de pares de inteiros. Os elementos de cada par representam o número de pessoas que embarcam no ônibus (o primeiro item) e o número de pessoas que desembarcam (o segundo item) em um determinado ponto de ônibus.

Sua tarefa é retornar o número de pessoas que ainda estão no ônibus após o último ponto (após o último elemento do array). Mesmo sendo o último ponto, o ônibus pode não estar vazio e algumas pessoas podem ainda estar dentro dele, provavelmente dormindo :D

Veja os casos de teste.

Lembre-se de que os casos de teste garantem que o número de pessoas no ônibus seja sempre >= 0. Portanto, o inteiro retornado não pode ser negativo.

O segundo valor do primeiro par no array é 0, pois o ônibus está vazio no primeiro ponto.
