# Regex validate PIN code

<p>Os caixas eletrônicos permitem códigos PIN de 4 ou 6 dígitos, e os códigos PIN não podem conter nada além de exatamente 4 ou exatamente 6 dígitos.</p>

<p>Se a função receber uma sequência PIN válida, retorne verdadeiro; caso contrário, retorne falso.</p>

## Examplos (Entrada --> Saída)

```
"1234"   -->  true
"12345"  -->  false
"a234"   -->  false
```
